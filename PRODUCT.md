# Product

## Register

brand

## Users

Visitors in Stratford-upon-Avon who want to decide quickly whether Havilands Tea Room suits a tea, breakfast, cream-tea, or light-lunch stop, then need phone, address, hours, and map details.

## Product Purpose

Create a polished single-page public website for Havilands Tea Room using real available photos and listing information. The page should make the place feel characterful, local, and easy to visit.

## Brand Personality

Traditional, cosy, slightly whimsical, and practical. The Alice in Wonderland detail can be present through imagery and copy, but the page should still feel like a usable local-business website.

## Content Principles

1. Use public facts that can be traced to the provided Facebook page and public listing pages.
2. Keep menu claims broad where the source menu has incomplete or inconsistent price fields.
3. Make phone, map, and visit information available above and below the fold.
4. Do not invent booking, delivery, reviews, awards, or official-owner claims.

## Accessibility

Use semantic landmarks, useful alt text, keyboard focus states, readable contrast, responsive image proportions, and reduced-motion support.
