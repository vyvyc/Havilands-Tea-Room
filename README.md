# Havilands Tea Room

A polished single-page website for Havilands Tea Room in Stratford-upon-Avon.

Live site: https://vyvyc.github.io/Havilands-Tea-Room/

## Source Material

- Facebook public page: https://www.facebook.com/p/Havilands-Tea-Room-61556942260846/
- GoToEat listing and photo source: https://havilandstearoom.gotoeat.net/
- GoToEat menu source: https://havilandstearoom.gotoeat.net/menu

## Build Notes

- Static HTML, CSS, and JavaScript.
- GSAP is vendored in `vendor/gsap.min.js` for first-load and scroll-triggered motion.
- Real public listing photos are used for the storefront, menu, and gallery.
- The garden seating image was regenerated from the supplied/public tea-room photo into a commercial photography treatment and optimized for the web.
